# Consulta PIX

Sistema para consulta de dados do PIX e CCS (Cadastro de Clientes do Sistema Financeiro) do BACEN.

## Tecnologias

- Backend: Go
- Frontend: React com TypeScript e Material UI

## Estrutura do Projeto

- `backend/`: API em Go
- `frontend/`: Interface de usuário em React
